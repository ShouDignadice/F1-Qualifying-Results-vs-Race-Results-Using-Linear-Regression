# Formula 1 Qualifying vs Race Results

## Project Objective

This project analyzes the relationship between Formula 1 qualifying results and actual race results.

The main goal is to understand whether a driver's qualifying position can help explain or predict their final race finishing position.

## Main Project Question

**Does qualifying position have a strong relationship with race finishing position in Formula 1?**

## Supporting Questions

- Do drivers who qualify near the front usually finish near the front?
- How much does starting position affect race outcome?
- Are qualifying results a good predictor of race results?
- How strong is the relationship between qualifying position and final finishing position?
- What does a simple linear regression model reveal about this relationship?

## Why This Project Matters

Formula 1 races are influenced by many factors, including driver skill, car performance, team strategy, pit stops, penalties, weather, and mechanical issues.

Even with those factors, qualifying position is often viewed as important because starting closer to the front gives drivers a better chance of finishing well.

This project uses data analysis and linear regression to test that idea using real Formula 1 data.
